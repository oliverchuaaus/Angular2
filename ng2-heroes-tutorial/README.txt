This is a reference app from the tutorial.

Reference: https://angular.io/docs/ts/latest/tutorial/

Prerequisites:
git installed
nodejs installed


Codebase:
download zip from plunker
replace files with files from quickstart minimal
add "angular-in-memory-web-api": "0.1.10", in package.json


Run in cmd:
cd ng2-heroes-tutorial
npm install
npm start

Check that it runs npm and shows page.
